# 11 — 网站设计规划

> 本文档定义芜湖旅游推荐平台的视觉设计规范、页面布局、组件样式及交互动效。
> 设计原则：**清新自然 + 高级感动效 + 无额外动画依赖库**。

---

## 一、色彩系统

> **配色策略**: Committed。以江蓝色为主色承载 40%+ 品牌面积，徽文化意象贯穿，拒绝通用 SaaS 蓝绿色盘。
> **灵感来源**: 长江江水（主色）、徽派水墨（文字）、皖南群山（自然绿）、芜湖小吃（暖橙）、夜景灯光（紫调）。

### 颜色使用约束

- 灰色文字不得出现在有色背景上；使用该色调的同色系或 `color-mix(in oklch, var(--color-ink) 50%, transparent)` 控制对比度。
- 文字-背景对比度 ≥ 4.5:1（AA 级），大号文字 ≥ 3:1。
- 占位文字同样需要 ≥ 4.5:1，不使用 `#9CA3AF` 类低对比度灰色。
- `#F6F8F8` 背景非暖灰：冷微色调指向江水意象，避免落入 AI 默认暖纸色板。

## 二、字体层级

字体栈：`--font-sans` → `"PingFang SC", "Microsoft YaHei", "Hiragino Sans GB", system-ui, -apple-system, sans-serif`

> 中文排版要点：行高比英文字体高 0.1-0.2；标题用 `text-wrap: balance` 平衡换行，正文用 `text-wrap: pretty` 减少孤字。

| 层级 | 大小 | 字重 | 行高 | `text-wrap` | 用途 |
|------|------|------|------|-------------|------|
| Display | 2.5-3.5rem (clamp) | 700 | 1.2 | `balance` | 首页 Hero 大标题 |
| H1 | 2.25rem (36px) | 700 | 1.3 | `balance` | 详情页标题 |
| H2 | 1.5rem (24px) | 600 | 1.35 | `balance` | 区块标题 |
| H3 | 1.125rem (18px) | 600 | 1.45 | — | 卡片标题 |
| Body L | 1rem (16px) | 400 | 1.7 | `pretty` | 正文描述（最大行宽 65ch） |
| Body | 0.9375rem (15px) | 400 | 1.6 | `pretty` | 一般正文 |
| Caption | 0.875rem (14px) | 400 | 1.5 | — | 地址、日期、提示文字 |
| Small | 0.8125rem (13px) | 500 | 1.4 | — | 标签、角标、分类名 |

### 排版规则

- **字重对比**：标题与正文之间字重差 ≥ 200，形成明确层级区分。
- **行宽上限**：正文 ≤ 65ch，列表/卡片无严格限制。
- **孤行控制**：段落末行不少于 3 个中文字或 5 个英文词。
- **大写约束**：仅用于 ≤4 词标签、角标、分类名。不做全大写正文。
- **字间距**：标题 `letter-spacing` ≥ -0.02em；不缩紧到字符粘连。

**Hero 设计要点**：
- **非对称分栏**：左 40% 文字区（深色渐变底 `#0E2D3B` → `#157878`），右 60% 轮播区。不再使用"大图叠大字"的模板套用。
- **徽派纹样装饰**：左侧文字上方用一条 2px 高的装饰线，分段式（3 段短横线），颜色 `--color-primary`，向徽派建筑的瓦当纹样致敬。
- **文字排版**：主标题 "芜湖，半城山水半城诗" 按自然断句分行，`font-size: clamp(2.5rem, 5vw, 3.5rem)`，`font-weight: 700`，`color: #FFFFFF`，`text-wrap: balance`。副标题 "江水穿城，徽韵千年" 用 `font-size: 1.125rem`，`opacity: 0.85`。
- **CTA 按钮**：`background: linear-gradient(135deg, #157878, #0E6060)`，内边距 `14px 36px`，字体 1rem 600 白色。`border-radius: 8px`。hover 时 `scale(1.04)` + `box-shadow` 扩散。
- **轮播指示器**：圆形小点，当前页圆点加宽 3 倍 + 颜色变亮。
- **移动端**：左右分栏变为上下叠放，轮播区 60vh，文字区填充剩余高度。

## 六、组件设计规范

### 6.1 NavBar 导航栏

| 属性 | 首页顶部（透明态） | 滚动后 / 其他页（实底态） |
|------|------------------|--------------------------|
| 背景 | `linear-gradient(to bottom, rgba(0,0,0,0.45), transparent)` | `#FFFFFF` |
| 文字色 | `#FFFFFF` | `var(--color-ink)` |
| Logo 色 | 白色 | `var(--color-primary)` |
| 阴影 | 无 | `0 2px 12px rgba(0,0,0,0.08)` |
| 高度 | 64px | 64px |
| 位置 | `position: fixed; top: 0; z-index: 100` | 同上 |

**导航项交互**：
- 默认：无下划线
- Hover：底部 2px 蓝色下划线从中间向两端展开（`scaleX 0→1`，0.3s）
- 当前路由：下划线保持显示

**移动端**：`< 768px` 时导航缩为汉堡菜单，点击展开垂直菜单（Element Plus Drawer）。

## 七、动画规范

### 7.1 动画总则

- 所有动画以 `transform` 和 `opacity` 为首选属性（GPU 加速）
- 避免动画 `width`、`height`、`top`、`left`（触发重排）
- 动画时长 0.2s-0.4s，不超过 0.6s
- 缓动函数：`cubic-bezier(0.4, 0, 0.2, 1)`（Material Design 标准缓出）
- 弹性动画：`cubic-bezier(0.34, 1.56, 0.64, 1)`（仅用于重点元素如 CTA 按钮）
- **无障碍必做**：所有动画须包装在 `@media (prefers-reduced-motion: no-preference)` 内
- 降级方案：`prefers-reduced-motion: reduce` 时使用 `opacity` 淡入淡出（≤ 0.1s）替代位移/缩放动画

```css
/* 默认无动画；由 prefers-reduced-motion: no-preference 开启 */
@media (prefers-reduced-motion: no-preference) {
  .animate-in {
    animation: fadeInUp 0.5s cubic-bezier(0.4, 0, 0.2, 1) forwards;
  }
}

@media (prefers-reduced-motion: reduce) {
  .animate-in { opacity: 1; }
}
```

### 7.2 页面级动画

#### 路由过渡（Vue `<Transition>`）

```css
.page-enter-active { transition: opacity 0.3s, transform 0.3s; }
.page-leave-active { transition: opacity 0.15s; }
.page-enter-from { opacity: 0; transform: translateY(20px); }
.page-leave-to   { opacity: 0; }
```

#### 首页 Hero 文字动画

| 元素 | 动画 | 延迟 | 时长 |
|------|------|------|------|
| 大标题 | 逐字上浮（每个字 stagger 0.08s） | 0.2s | 0.4s/字 |
| 副标题 | 整体 fade-in + slide-up 20px | 1.0s | 0.5s |
| CTA 按钮 | 缩放弹入 scale(0→1.1→1) | 1.5s | 0.4s |

实现方式：用 CSS `@keyframes` + `animation-delay` 给每个 `<span>` 设置递增延迟。

#### 滚动入场（IntersectionObserver）

```javascript
// 核心逻辑伪代码
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.classList.add('animate-in');
      observer.unobserve(entry.target); // 只触发一次
    }
  });
}, { threshold: 0.15 });
```

**CSS 类**：

```css
.animate-in {
  animation: fadeInUp 0.5s cubic-bezier(0.4, 0, 0.2, 1) forwards;
}

@keyframes fadeInUp {
  from { opacity: 0; transform: translateY(40px); }
  to   { opacity: 1; transform: translateY(0); }
}
```

**Stagger 延迟**：
- 分类入口 4 卡片：`0s, 0.12s, 0.24s, 0.36s`
- 热门推荐卡片：每行 `0.08s` 递增
- 列表页卡片：每张 `0.06s` 递增

### 7.3 组件动画

| 组件 | 触发条件 | 动画 |
|------|----------|------|
| NavBar | 滚动位置变化 | 背景透明度渐变 `0.3s`，文字颜色渐变 |
| 卡片 | `:hover` | `translateY(-6px)` + 阴影扩散 `0.3s` + 图片 `scale(1.05)` `0.35s` |
| 搜索框 | `:focus` | 宽度 `240px → 360px`（桌面），`0.4s` |
| 分类菜单 | `:hover` | 左侧 3px 蓝色竖线滑入，`0.15s` |
| 评分星 | 页面加载 | 从左到右依次点亮，每星 `0.15s` stagger |
| CTA 按钮 | `:hover` | `scale(1.04)` `0.2s`；`:active` `scale(0.97)` |
| 回顶部按钮 | 滚动 > 500px | 从右下角 `rotate(90deg)` + `fadeIn` 进入 `0.3s` |
| 加载态 | 数据请求中 | 骨架屏 `pulse` 呼吸动画，循环 `1.5s` |
| 空状态图标 | 显示时 | 轻微上下浮动 `bounce`，循环 `2s` |

### 7.4 AI 聊天窗动画

| 触发 | 动画描述 | 参数 |
|------|----------|------|
| 打开 | 从按钮位置缩放展开 (`transform-origin: bottom right`) | `scale(0→1)` + `opacity(0→1)`，`0.25s` |
| 关闭 | 缩小收回 + 淡出 | 同上反向，`0.2s` |
| 新消息 | 从底部 `20px` 滑入 + 淡入 | `0.3s` ease |
| 思考中 | 三个点依次上下跳动 (bounce) | 每点延迟 `0.2s`，`0.6s` 循环 |
| 快捷按钮 | `:hover` 上浮 `2px` + 边框变蓝 | `0.2s` |
| 发送按钮 | 有内容时灰色→蓝色渐变；`:active` `scale(0.92)` | `0.3s` / `0.1s` |

**思考中动画 CSS**：

```css
@keyframes bounce-dot {
  0%, 80%, 100% { transform: translateY(0); }
  40% { transform: translateY(-6px); }
}
.dot:nth-child(1) { animation-delay: 0s; }
.dot:nth-child(2) { animation-delay: 0.2s; }
.dot:nth-child(3) { animation-delay: 0.4s; }
```

### 7.5 视差效果

| 位置 | 效果 | 参数 |
|------|------|------|
| 首页轮播图背景 | 滚动时图片以 `0.5x` 倍速移动 | `background-attachment: fixed` 或 JS `transform: translateY(scrollY * 0.5)` |

## 十三、CSS 架构与主题系统

### 设计 Token 分层

所有视觉属性通过 CSS 自定义属性定义，分三层：

```
Layer 1: 基础色值 (--color-*, --text-*, --radius-*, --shadow-*)
Layer 2: 语义映射 (--brand-*, --gradient-*, --color-success/danger/warning)
Layer 3: 组件绑定 (组件内通过 var() 引用 Layer 1/2 的 token)
```

### 暗色模式预备

当前优先实现亮色模式。预留暗色模式切换入口：

```css
:root {
  color-scheme: light;
}

[data-theme="dark"] {
  color-scheme: dark;
  /* 暗色模式 token 覆盖在此定义，后续迭代补齐 */
  --color-bg:    #111820;
  --color-surface: #1A2330;
  --color-ink:      #E8ECF0;
  --color-ink-secondary: #9AACBD;
  --color-ink-muted: #6A7A8A;
  --color-border: #2A3540;
}
```

### 全局样式入口

```css
/* global.css */
@import './tokens.css';

}
```

## 十四、无障碍规范 (a11y)

### 键盘导航

| 元素 | 操作 | 要求 |
|------|------|------|
| 导航链接 | `Tab` 切换 | 每个 `<a>` / `<button>` 可聚焦，`focus-visible` 轮廓可见 |
| 卡片列表 | `Tab` 逐卡移动 | 卡片包装为 `<a>` 链接，确保能通过键盘选中 |
| 分类入口 | `Tab` 聚焦 | 每个分类卡可聚焦，点击/Enter 跳转 |
| 搜索框 | `Tab` 进入，`Escape` 退出 | 输入实时触发搜索建议 |
| 轮播图 | `← →` 箭头切图 | 提供 `<button>` 左右箭头，`aria-label` 标注 |
| 聊天窗 | `Escape` 关闭，`Tab` 在消息区内流转 | 消息区 `role="log"`，新消息用 `aria-live="polite"` 播报 |
| 汉堡菜单 | `Tab` 到按钮，`Enter` 展开/合拢 | `aria-expanded` 状态切换 |

### 屏幕阅读器适配

| 元素 | ARIA 属性 | 说明 |
|------|-----------|------|
| 导航栏 | `<nav aria-label="主导航">` | 区分导航区域 |
| 轮播图 | `role="region" aria-roledescription="carousel" aria-label="芜湖风景轮播"` | 标注轮播区 |
| AI 消息列表 | `role="log" aria-live="polite" aria-label="对话消息"` | 新消息自动播报 |
| 图片 | 必须提供 `alt` 属性 | 景点图写具体名称，纯装饰图 `alt=""` |
| 评分星 | `aria-label="评分 4.5 / 5"` | 使用 Element Plus Rate 内置 ARIA |
| 空状态 | `role="status"` | 告知当前无数据 |
| 加载态 | `aria-busy="true"` + `aria-live="polite"` | 告知数据加载中 |

### Focus 可见性

所有可交互元素必须具备 `:focus-visible` 样式：
- 默认：2px 实线边框 `outline: 2px solid var(--color-primary); outline-offset: 2px`
- 暗色背景上的焦点环使用白色：`outline-color: #FFFFFF`
- 移除 Element Plus 默认的 `box-shadow` 聚焦样式，改用标准 `outline`

### 色彩对比度验证

| 关系 | 最小对比度 | 验证点 |
|------|-----------|--------|
| 正文与背景 | 4.5:1 | `--color-ink` vs `--color-bg` |
| 大号文字（≥18px bold）与背景 | 3:1 | 标题 vs 背景 |
| 占位文字与背景 | 4.5:1 | search placeholder vs input bg |
| 按钮文字与按钮背景 | 4.5:1 | 白色文字 vs 江蓝渐变底 |
| 卡片遮罩文字 | 3:1 | 白色文字 vs 渐变遮罩（因大号文字） |

> 建议使用 Chrome DevTools 的 CSS Overview 面板或 axe DevTools 逐页检测。

## 十五、交互状态规范

所有可交互组件必须覆盖以下状态：

| 状态 | 触发条件 | 要求 |
|------|----------|------|
| **默认** | 初始渲染 | 视觉层级正确，不抢夺焦点 |
| **:hover** | 鼠标悬停 | 色值变化 / 阴影变化 / 微缩（任一，不要叠用） |
| **:focus-visible** | 键盘 Tab 聚焦 | 可见 outline，不与 hover 混淆 |
| **:active** | 鼠标按下 / 触摸 | `scale(0.97)` 或颜色加深，反馈即时 |
| **禁用** | `disabled` 属性 | `opacity: 0.5` + `cursor: not-allowed` + 移除 hover 效果 |
| **加载中** | 异步操作进行中 | 按钮内 spinner + 文字替换为 "处理中..."，禁止重复提交 |
| **错误** | 输入验证失败 | 红色边框 + 行内错误文案，不在顶部弹 toast |
| **成功** | 操作完成 | 短暂成功图标（≤ 2s）+ 自动恢复 |
