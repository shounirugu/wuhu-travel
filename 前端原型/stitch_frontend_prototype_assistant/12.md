## 一、技术选型：ECharts + vue-echarts

### 为什么选 ECharts

| 维度 | ECharts | Chart.js | D3.js |
|------|---------|----------|-------|
| 配置驱动 vs 代码驱动 | 配置驱动，声明式，改起来快 | 命令式，偏代码 | 代码驱动，最底层 |
| 图表丰富度 | 丰富（饼/柱/折/地图/散点…） | 中等 | 最高（但需手写） |
| Vue 3 集成 | vue-echarts 官方封装，响应式开箱即用 | 手动管理 canvas 生命周期 | 同上 |
| 动画质量 | 内置高质量入场/更新动画 | 基础动画 | 自定义 |
| 学习成本 | 中（配置项多，但文档好） | 低 | 高 |
| 国内文档/社区 | 极丰富 | 一般 | 一般 |
| **简历价值** | **国内主流，面试官认可度高** | 一般 | 高但复杂 |

**结论**：ECharts + vue-echarts 是本项目最合适的选择。

### 安装

```bash
npm install echarts vue-echarts
```

### 全局注册（main.js）

```javascript
import { createApp } from 'vue'

```

## 三、图表详细设计规范

### 3.1 分类分布 — 环形饼图（CategoryPieChart.vue）

**图表类型**：ECharts Pie（环形，`radius: ['45%', '70%']`）

**配色**（与首页分类卡片严格一致）：

| 分类 | 颜色 | 色值 |
|------|------|------|
| 风景 | 蓝色 | `#4A90D9` |
| 项目 | 绿色 | `#52B788` |
| 小吃 | 橙色 | `#F5A623` |
| 打卡地 | 紫色 | `#8B5CF6` |

**ECharts option 核心配置**：

```javascript
const option = computed(() => ({
  tooltip: {
    trigger: 'item',
    formatter: '{b}: {c} 个 ({d}%)',
  },
  legend: {
    orient: 'vertical',
    right: '5%',
    top: 'center',
    textStyle: { color: '#1F2937', fontSize: 13 },
  },
  series: [
    {
      name: '景点分类',
      type: 'pie',
      radius: ['45%', '70%'],
      center: ['40%', '50%'],
      avoidLabelOverlap: false,
      label: { show: false },           // 环内不显示标签
      labelLine: { show: false },
      emphasis: {
        label: { show: true, fontSize: 16, fontWeight: 'bold' },
        itemStyle: { shadowBlur: 10, shadowOffsetX: 0, shadowColor: 'rgba(0,0,0,0.2)' },
      },
      data: props.data.map((item, i) => ({
        name: item.categoryName,
        value: item.count,
        itemStyle: { color: CATEGORY_COLORS[i] },
      })),
    },
  ],
}));
```

### 3.2 评分排行 — 水平条形图（RatingBarChart.vue）

**图表类型**：ECharts Bar（horizontal，`yAxis.type: 'category'`）

**条形颜色**：按目的地所属分类，使用对应分类色（与饼图配色统一）

**ECharts option 核心配置**：

```javascript
const option = computed(() => ({
  tooltip: {
    trigger: 'axis',
    axisPointer: { type: 'shadow' },
    formatter: (params) => {
      const d = params[0];
      return `${d.name}<br/>评分：${d.value} ⭐`;
    },
  },
  grid: { left: '20%', right: '10%', top: '5%', bottom: '5%', containLabel: true },
  xAxis: {
    type: 'value',
    min: 4.0,
    max: 5.0,
    axisLabel: { formatter: (v) => v.toFixed(1) },
    splitLine: { lineStyle: { type: 'dashed', color: '#E5E7EB' } },
  },
  yAxis: {
    type: 'category',
    data: props.data.map(d => d.name).reverse(),   // 高分在上
    axisLabel: { fontSize: 12, color: '#1F2937' },
  },
  series: [
    {
      type: 'bar',
      barMaxWidth: 24,
      label: {
        show: true,
        position: 'right',
        formatter: '{c}',
        color: '#6B7280',
        fontSize: 12,
      },
      data: props.data.map(d => ({
        value: d.rating,
        itemStyle: { color: CATEGORY_COLOR_MAP[d.categoryName], borderRadius: [0, 4, 4, 0] },
      })).reverse(),
    },
  ],
}));
```

### 3.3 AI 对话趋势 — 折线图（AiTrendChart.vue）

**图表类型**：ECharts Line（smooth: true，area 填充）

**数据来源**：前端 Mock 静态数据，不请求后端

**三条折线颜色**：

| 功能 | 颜色 |
|------|------|
| 智能问答 | `#4A90D9`（主色蓝） |
| 路线规划 | `#52B788`（辅色绿） |
| 个性化推荐 | `#F5A623`（点缀橙） |

**ECharts option 核心配置**：

```javascript
const option = computed(() => ({
  tooltip: {
    trigger: 'axis',
    axisPointer: { type: 'cross', label: { backgroundColor: '#6a7985' } },
  },
  legend: {
    data: ['智能问答', '路线规划', '个性化推荐'],
    top: 0,
    textStyle: { color: '#6B7280' },
  },
  grid: { left: '3%', right: '4%', bottom: '3%', top: '40px', containLabel: true },
  xAxis: {
    type: 'category',
    boundaryGap: false,
    data: aiStatsData.dates,
    axisLine: { lineStyle: { color: '#E5E7EB' } },
    axisLabel: { color: '#6B7280' },
  },
  yAxis: {
    type: 'value',
    splitLine: { lineStyle: { type: 'dashed', color: '#E5E7EB' } },
    axisLabel: { color: '#6B7280' },
  },
  series: aiStatsData.series.map((s, i) => ({
    name: s.name,
    type: 'line',
    smooth: true,
    symbol: 'circle',
    symbolSize: 6,
    lineStyle: { width: 2.5, color: LINE_COLORS[i] },
    itemStyle: { color: LINE_COLORS[i] },
    areaStyle: {
      color: {
        type: 'linear', x: 0, y: 0, x2: 0, y2: 1,
        colorStops: [
          { offset: 0, color: hexToRgba(LINE_COLORS[i], 0.3) },
          { offset: 1, color: hexToRgba(LINE_COLORS[i], 0.02) },
        ],
      },
    },
    data: s.data,
  })),
}));
```

## 七、后端路由实现（server/routes/stats.js）

```javascript
// server/routes/stats.js
const express = require('express');
const router = express.Router();
const db = require('../db');

router.get('/', async (req, res, next) => {
  try {
    // 1. 概览数据（总数 + 均分 + 分类数）
    const [[overview]] = await db.execute(`
      SELECT
        COUNT(*) AS totalDestinations,
        ROUND(AVG(rating), 1) AS avgRating,
        (SELECT COUNT(*) FROM categories) AS totalCategories
      FROM destinations
    `);

    // 2. 分类分布（用于饼图）
    const [categoryDistribution] = await db.execute(`
      SELECT c.id AS categoryId, c.name AS categoryName, COUNT(d.id) AS count
      FROM categories c
      LEFT JOIN destinations d ON c.id = d.category_id
      GROUP BY c.id, c.name
      ORDER BY c.sort_order
    `);

    // 3. Top 10 评分排行（用于条形图）
    const [topRatedDestinations] = await db.execute(`
      SELECT d.id, d.name, d.rating, c.name AS categoryName
      FROM destinations d
      JOIN categories c ON d.category_id = c.id
      ORDER BY d.rating DESC
      LIMIT 10
    `);

    res.json({
      code: 200,
      message: 'success',
      data: { overview, categoryDistribution, topRatedDestinations },
    });
  } catch (err) {
    next(err);
  }
});

module.exports = router;
```

**在 index.js 中注册路由**：

```javascript
// server/index.js（新增一行）

```
